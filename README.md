# Holberton School - Interview Preparation

## Description
This repository contains coding challenges and algorithmic problems commonly encountered during technical interviews. Each directory focuses on a specific topic or problem type, providing implementations and solutions in Python.

## Structure
Each project directory contains:
- Implementation files with solutions
- Test files for validation
- Individual README with problem-specific details

## Projects
- **lockboxes/**: Algorithm to determine if all locked boxes can be opened

## Requirements
- **Language**: Python 3 (version 3.4.3+)
- **Environment**: Ubuntu 14.04 LTS
- **Style**: PEP 8 compliant
- **Editors**: vi, vim, emacs

## Usage
Navigate to the specific project directory and run the test files:
```bash
cd [project_directory]
./main_[number].py
```
